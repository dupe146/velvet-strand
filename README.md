# Velvet Strand — Luxury Hair Studio Website

A fully responsive, multi-page website built for a luxury hair salon brand based in Lagos, Nigeria.

## Live Site
[velvet-strand.vercel.app](https://velvet-strand.vercel.app)

## Built With
- React 18 (Vite)
- React Router v6
- CSS (custom properties, no UI library)
- Formspree (booking form)
- Deployed on Vercel

## Features
- Mobile-first responsive design
- Multi-page routing (Home + Gallery)
- Booking form with email notifications
- WhatsApp click-to-chat integration
- Image gallery with filter tabs and lightbox
- Scroll-triggered animations
- Custom brand colour palette

## Pages
- **Home** — Hero, Services, About, Gallery preview, Testimonials, Booking form
- **Gallery** — Filterable masonry grid with full-screen lightbox
